### About

Allegra is a blockchain explorer and UI for interacting with the Algorand blockchain. It allows you to connect your wallet, compose and send basic transactions, and search for transactions, assets, accounts, applications and other information stored on-chain.

### Involvement

Creator and maintainer.

### Stack

**Design**: Figma

**Frontend**: React, TypeScript, Next.js, Jest, ChakraIU/Emotion, Redux Toolkit

**CI/CD**: Github Actions, Vercel
